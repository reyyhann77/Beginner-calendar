# Beginner-calendar
My first interactive calendar made with HTML, CSS, and JavaScript


A simple interactive calendar built with **HTML, CSS, and JavaScript**.  
This is my first small project while learning front-end development and Electron.

---

## Features
- Displays a monthly calendar dynamically.
- Highlights current date.
- Simple hover effects for interaction.
- Fully static, lightweight, and beginner-friendly.

beginner-calendar/

├─ main.js           # Electron main process
├─ script.js         # Renderer JS

├─ package-lock.json
